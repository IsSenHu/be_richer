## JVM参数

| 参数                             | 作用                    |
| -------------------------------- | ----------------------- |
| -XX:+&lt;option&gt;              | 表示开启option选项      |
| -XX:-&lt;option&gt;              | 表示关闭option选项      |
| -XX:&lt;option&gt;=&lt;value&gt; | 表示给option赋值为value |
| -XX:+TraceClassLoading           | 跟踪类的加载            |
| -XX:+TraceClassUnloading         | 跟踪类的卸载            |
